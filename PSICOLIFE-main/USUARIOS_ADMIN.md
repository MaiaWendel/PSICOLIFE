## 🚀 COMO USAR

### 1. Execute o Backend:
```bash
cd Backend
python App.py
```

### 2. Acesse as páginas:
- **Site Principal**: Abra `index.html`
- **Admin Dashboard**: `assets/html/admin-dashboard.html`
- **Login Admin**: `assets/html/admin-login.html`

### 3. Para criar os usuários no banco:
```bash
python Backend/Init_DB.py
```
